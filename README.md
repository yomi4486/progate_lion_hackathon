# Alive Talk - 人とAIをライブで繋ぐサービス

### リポジトリをクローン    
```sh
git clone https://github.com/yomi4486/progate_lion_hackathon.git
cd progate_lion_hackathon
```
### フロントエンドの起動

#### ios
```sh
cd frontend && make ios
```

#### web
```sh
cd frontend && make web
``` 
### バックエンドの起動    
```sh
cd backend
yarn install
yarn run dev
yarn run migrate
```
